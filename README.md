# Parental-Legacy
A mern stack project of random values.

# Step-by-Step Procedure to Run:
1. Install Node.js, and React.
2. Install packages of react and nodejs.
3. Run 'npm install' in both frontend and backend folders in VS Code.
4. Locate to 'frontend' folder in terminal and run 'npm run dev'.
5. Locate to 'backend' folder in terminal and run 'npm start'.
6. You will see home page with table.
